Errors running builder 'Integrated External Tool Builder' on project

分类： J2EE 2012-05-17 14:40 2119人阅读 评论(0) 收藏 举报
myeclipseproperties

在网上下载到一个开源项目后，导入MyEclipse，编译工程的时候遇到下面的错误：

Errors running builder 'Integrated External Tool Builder' on project 'patrol'

解决方法是：

右键项目选择“Properties”，再选择“Builders”，删除丢失的builder，我这里是“Builder ExternalToolBuilder”。
